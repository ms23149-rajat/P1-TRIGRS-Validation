## Notes

Current TRIGRS runs use Srivastava & Yeh (1991) benchmark parameters.

These values must be replaced with Wayanad-specific values before realistic rainfall-triggered simulations can be performed.

All adopted values must be traceable to published literature, reports, field measurements, or authoritative datasets.
