# Wayanad TRIGRS Raw Results

## Purpose

This directory documents the raw outputs generated during the Wayanad TRIGRS implementation and validation workflow.

---

## Primary Raw Output

File:

```text
TRlist_z_p_fs_wayanad.txt
```

Generated by:

```text
TRIGRS
```

Description:

Contains pressure head and Factor of Safety (FoS) values for all cells and depths computed during the simulation.

## Notes

The current Wayanad run uses:

* Copernicus GLO-30 DEM
* Srivastava & Yeh (1991) benchmark soil parameters
* No rainfall forcing
* No runoff-routing files

The results should therefore be interpreted as workflow-validation outputs rather than realistic landslide susceptibility estimates.
